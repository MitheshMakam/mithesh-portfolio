# Mithesh Portfolio

A FAANG-ready developer portfolio built with React.

## Quick Start

```bash
npm install
npm start
```

## Deploy to Vercel (Free)

1. Push this folder to a GitHub repo
2. Go to https://vercel.com → Sign up with GitHub
3. Click "Add New Project" → Import your repo
4. Click "Deploy" — done in 60 seconds!

Your live URL will be: https://mithesh-portfolio.vercel.app

## Customize

- Update your real project links in `src/App.jsx`
- Replace placeholder emails/LinkedIn/GitHub URLs
- Add your actual `resume.pdf` into the `public/` folder
